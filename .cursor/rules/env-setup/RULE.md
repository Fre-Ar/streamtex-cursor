# Environment & Execution

## Shell Configuration
If asked to run code, verify the environment first:

1. **Activate Environment:**
```bash
conda activate streamtex
```

2.  **Run the Book:**
Navigate to the project folder and run:
```bash
streamlit run book.py
```

## Dependencies
- If import errors occur, check `streamtex_package/requirements.txt`.
- Ensure `streamtex_package` is located in the parent directory of your project folder.
